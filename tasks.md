# K12 企业级教务管理系统 - 任务分解 (Tasks)

## 1. 基础设施搭建 (P0)
- [ ] **项目初始化**: 创建 monorepo 仓库，包含前端 (client) 和后端 (server)。
- [ ] **数据库设计**: 使用 PostgreSQL，设计 User, Role, Permission, Student, Teacher, Grade, Class 表。
- [ ] **RBAC 实现**: 基础的认证与授权中间件（JWT + 角色守卫）。
- [ ] **低代码核心**: 实现 Schema 定义与简单的 CRUD API 生成器。

## 2. 核心业务模块 (P1)
- [ ] **用户管理**:
    - [ ] 批量导入学生/教师数据（Excel解析）。
    - [ ] 动态分配角色（年级组长、班主任）。
- [ ] **学期与科目配置**:
    - [ ] 增加新学期、新科目（动态添加）。
    - [ ] 绑定年级与科目的关系（高一开物理，高二开化学）。
- [ ] **考试与成绩管理 (难点)**:
    - [ ] 创建一次考试（配置参与科目、满分）。
    - [ ] 录入成绩（Excel导入/在线录入）。
    - [ ] **成绩计算引擎**: 自动计算班级排名、年级排名、进退步。

## 3. 低代码扩展 (P2)
- [ ] **表单设计器**:
    - [ ] 拖拽组件（文本、数字、日期、图片）。
    - [ ] 发布表单并生成数据收集链接。
- [ ] **流程引擎 (简易版)**:
    - [ ] 定义审批节点（班主任 -> 年级组长）。
    - [ ] 发起审批申请。

## 4. 前端展现 (P3)
- [ ] **管理后台**: Ant Design Pro 布局。
- [ ] **数据大屏**: 简单的 ECharts 图表展示（班级均分对比）。
- [ ] **移动端适配**: 简单的 H5 页面供学生/家长查分。

# K12 企业级教务管理系统 - 验收清单 (Checklist)

## 1. 基础架构与权限
- [ ] 数据库 Schema 已创建，User/Role/Permission 关系正确。
- [ ] 系统管理员可以登录，并能创建新用户。
- [ ] 不同角色登录后看到的菜单不同（权限控制生效）。

## 2. 成绩与考务
- [ ] 管理员能创建一个新的考试批次（如：2025春季期中考）。
- [ ] 能导入一份 Excel 成绩表，数据正确存入数据库。
- [ ] 导入成绩后，系统自动计算出了“年级排名”和“班级排名”。
- [ ] 班主任登录后，能看到且**仅能看到**自己班级的成绩分析图表。

## 3. 低代码能力
- [ ] 管理员在后台能拖拽创建一个“秋游报名表”。
- [ ] 生成一个链接/二维码，普通用户能填写该表单。
- [ ] 管理员能在后台看到收集到的报名数据。
